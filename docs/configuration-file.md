# Configuration file

PresentPy can be configured externally using a configuration file. The configuration file is a TOML file that can be used to further customize the slides generated by PresentPy.

## Configuration file location

The configuration file can be placed in the following locations:

 - `presentpy.toml` in the current directory
 - `presentpy.toml` in the user's home directory
 - `.presentpy.toml` in the user's home directory

They are searched in the order above, so if you have a configuration file in the current directory, it will be used instead of the one in the user's home directory.

## Configuration options

```toml
font-size = 16

[highlight]
font-size = 18
bold = true
```
